# is-web-y27-lab-4

Одностраничное приложение на React для просмотра текущей погоды, 5-дневного прогноза и качества воздуха с поиском города.

## Инструкция по запуску
В корне репозитория выполнить комады:
`npm install`

`npm run build`

`npm run dev` — после этого в консоли появится url, по которому нужно перейти на сайт
